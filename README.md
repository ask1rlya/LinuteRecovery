# 🌿 Linute Recovery Project

**Linute** (Linux + Mint) is a custom recovery environment designed for a smooth, stable, and "soft" user experience. Our goal is to provide a reliable tool for device maintenance and custom firmware installation.

## 📱 Device Specifications: Redmi 10C
- **Codename:** fog / rain / wind
- **Chipset:** Qualcomm Snapdragon 680 (SM6225)
- **Kernel Arch:** arm64
- **Status:** In Development (Initial stage)
